## Features

⚡️ Nest\
⚡️ Docker\
⚡️ MongoDB and Mongoose

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

### Run DB

```bash
docker compose up --build -d
```

### Populate DB

```bash
# HTTP Get request

curl http://localhost:3000/api/v1/seed
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
